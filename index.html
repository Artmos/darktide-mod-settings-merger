Darktide Mod Settings Merger

Export or merge selected entries inside mods_settings from Warhammer 40,000: Darktide's user_settings.config. Everything runs locally in your browser.

Close Darktide before replacing user_settings.config. Keep a backup of your original file. Steam path: %AppData%\Fatshark\Darktide\user_settings.config. Microsoft Store path: %AppData%\Fatshark\MicrosoftStore\Darktide\user_settings.config.

1. Export selected mod settings

Choose your current user_settings.config, select the mods you want to share, then export a small file containing only those entries inside mods_settings.

2. Import / merge selected mods

Choose your current full config, then choose or paste someone else's shared mod settings. Select which shared mods to merge. Existing matching mods are replaced; missing mods are added.
